Site Name

Riot eEvents | LoL & Valorant

Site Purpose

This website provides information about current and upcoming e-sports events for the games League of Legends and Valorant. Visitors will find short descriptions about each game, event details, a countdown to the next tournament, and links to download the games directly from their official websites.

  • Where can I download them?
